# **App Name**: Die Zillertalerin - Frühstück

## Core Features:

- Menu Display: Display a menu of available breakfast items with descriptions and prices.
- Order Selection: Allow users to select items and specify quantities for their order.
- Order Summary: Provide a summary of the order, including total cost, before submission.
- Order Notification: Send an email notification to the apartment house owner upon new order submission.

## Style Guidelines:

- Primary color: Light, warm beige (#F5F5DC) to evoke a cozy morning feeling.
- Secondary color: Soft gray (#D3D3D3) for backgrounds and subtle contrasts.
- Accent: Burnt orange (#790000) to highlight key elements and call-to-actions.
- Clean, minimalist layout with ample spacing for a modern aesthetic.
- Simple, line-based icons for menu categories and order actions.
- Subtle transitions and animations for a smooth user experience.
